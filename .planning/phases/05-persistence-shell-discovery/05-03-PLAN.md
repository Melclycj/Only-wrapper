<objective>
The shell-discovery + dormant-UI vertical slice. After this plan the restored sessions from
Plan 05-02 are fully usable: the user sees a real shell DROPDOWN (not free-text) populated
from the OS, a dormant session shows an idle CARD with a ▶ Start button instead of a confusing
blank/dead terminal, an empty app shows a warm welcome CTA instead of auto-spawning a default
session, and the sidebar row offers a Start (▶) affordance that flips to Restart (↻) once the
session has run. Collapse state now persists.

Pieces (all renderer except the IPC already wired in Plan 05-01):
1. **Shell dropdown (D-05/SC4)** — `SessionEditModal.tsx`'s free-text shell input becomes a
   native `<select className="edit-select">` populated from `window.api.discoverShells()`;
   `$SHELL` is always present + default-selected; while in flight a disabled "Finding shells…"
   option shows. No free-text path (removes the renderer's ability to inject an arbitrary path).
2. **IdleCard (D-04)** — a new placeholder card rendered in `.terminal-area` when the active
   session is `not_started`: identity (icon+name+Idle badge), a read-only mono config block
   (cwd / shell / startupCommand — DISPLAYED, never executed — the TERM-05 boundary cue), and a
   ▶ Start session button that fires the existing ptyCreate/promote path (no new IPC).
3. **WelcomeEmptyState (D-10)** + **SessionManager boot rewrite** — replace the auto-add-on-empty
   boot + the reconcile poll with: one-shot listSessions snapshot, sort by order, focus first
   (D-09); zero sessions → WelcomeEmptyState; Start wiring (dormant row ▶ / IdleCard ▶ →
   ptyCreate with the dormant id → promote); collapse toggle mirrored via persistUiState (D-12).
4. **Sidebar Start/Restart label flip (D-03)** — the existing non-running Restart control becomes
   ▶ Start for `not_started` and ↻ Restart once it has run; the ContextMenu item flips the same way.

Purpose: make restored/dormant sessions a coherent, launchable experience and satisfy SC4 shell
discovery on macOS, behind the Windows-deferred seam.

Output: shell dropdown, IdleCard, WelcomeEmptyState, rewritten boot, Start/Restart flip, smoke.
</objective>
