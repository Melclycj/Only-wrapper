| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06.1-01 | DoS | `classify()` regexes over PTY output | mitigate | All regexes anchored (`\s*$`, `^...$/gm`) and linear — no nested quantifiers; input is a `rows×cols`-bounded viewport line array, never a growing byte stream (ASVS V5).
