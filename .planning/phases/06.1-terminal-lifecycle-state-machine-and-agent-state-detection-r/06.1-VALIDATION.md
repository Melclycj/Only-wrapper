## Sampling Rate

- **After every task commit:** Run `npm run test:unit` (Vitest — unit + offline replay).
- **After every plan wave:** Run the affected `tests/smoke/*.smoke.test.ts` specs.
- **Before `/gsd-verify-work`:** Full Vitest + full smoke suite green.
- **Phase gate:** full suite green → human-verify (the A1 amber-feel + Q1 scrollback / scroll-wheel checks from 06-04's checkpoint, re-run on the corrected model) → flip `06-VALIDATION.md` `nyquist_compliant: true` (Phase 6 sign-off this phase finishes).
- **Max feedback latency:** unit < ~10s; classifier sampling tick 100ms (settle T≈400–600ms) — do NOT sample slower than ~250ms or short agent pauses are missed (Nyquist bound from spike 002).
